Marvellous Restaurant Record App is a frontend web application developed using Angular and TypeScript that provides a structured platform for managing restaurant information. The application implements user authentication (Sign Up & Login) to ensure secure access to restaurant records.

After authentication, users can add, view, and manage restaurant details through a clean and intuitive interface. Restaurant data such as name, email, address, contact number, and services offered are captured using Angular forms and displayed in a tabular format.
